# Empório da Música Support Agent

Python customer-support agent built for the Artefact AI Engineer technical challenge.

The implementation combines typed tools over SQLite for operational data with a small
retrieval pipeline over the store policy PDF. Complete setup instructions, architecture
notes, limitations, and example conversations will be added as the implementation progresses.

## Planned commands

```bash
uv run emporio-setup
uv run emporio-chat
uv run streamlit run app.py
```

See [`docs/architecture.md`](docs/architecture.md) and
[`docs/execution-plan.md`](docs/execution-plan.md) for the approved implementation design.
